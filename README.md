## Getting Started

1. **Clone the repository:**

   ```bash
   git clone https://github.com/dominiksmolinski3/ProgramowanieAplikacjiWWW.git
   ```
2. **Navigate to the project directory:**

   ```
   cd ProgramowanieAplikacjiWWW
   ```

3. **Set up a local development environment:**

* Install a web server (e.g., Apache) with PHP support.
* Set up a MySQL database server.
* Configure the web server to serve files from the project directory.

4. **Import the database schema:**
* Locate the SQL files in the respective lab directories.
* Use a tool like phpMyAdmin or the MySQL command line to import the schema.

5. **Run the application:**
* Access the application through your web browser at http://localhost/{lab_directory}.
